Garnet installed

Rust Rigor. Ruby Velocity. One Coherent Language.

The garnet CLI is ready at /usr/local/bin/garnet.

Open a new Terminal window and run garnet --version to see the wordmark. Then start your first project with garnet new --template cli my_app.

Documentation lives at https://garnet-lang.org/docs. Research papers + Mini-Spec v1.0 ship in the monorepo under Garnet_Final/A_Research_Papers/ and Garnet_Final/C_Language_Specification/.
