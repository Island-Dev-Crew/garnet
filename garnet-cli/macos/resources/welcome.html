GARNET

Rust Rigor. Ruby Velocity. One Coherent Language.

Garnet is a dual-mode, agent-native language platform. Managed mode (def + ARC + exceptions) feels Ruby-like. Safe mode (@safe + fn + ownership + Result) feels Rust-like. The mode boundary auto-bridges errors and ownership.

This installer places the garnet CLI at /usr/local/garnet/bin/garnet with a symlink in /usr/local/bin/, so garnet --version resolves from any shell immediately after install.

After install, try:

  • garnet --version — confirm the toolchain is on PATH
  • garnet new --template cli my_app — scaffold a starter project
  • garnet run my_app/src/main.garnet — execute it
