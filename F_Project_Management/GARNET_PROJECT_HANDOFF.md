## THE VISION

Jon is ideating **Garnet** — a new programming language that reconciles **Rust's compile-time safety** with **Ruby's expressive beauty**. Core thesis: Rust and Ruby's weaknesses map with remarkable precision to each other's strengths, creating a rare opportunity for genuine language synthesis (not compromise).

**Garnet's design philosophy:** "Safe by default, fast when needed, joyful always."
**Dual-mode architecture:**
- **Managed mode** (default) — ARC memory, dynamic types, Ruby-like syntax, first-class REPL, performance comparable to Go
- **Safe mode** (`@safe` per-module) — Ownership + borrowing, static types with lifetimes, zero-cost abstractions, Rust-level performance

**Three target domains:** systems/performance + web/backend + scripting/automation
**Novel differentiator:** LLM-native syntax design for superior AI code generation
