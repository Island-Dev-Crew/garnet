# Archived Example Drafts

These files are historical design artifacts, not current runtime proof.

Use active examples under [`../../examples/`](../../examples) for current parse/check/run
evidence. Use this archive only to understand the original intended scale and
domain ambition of the MVP corpus.

## Contents

- `mvp-design-drafts/` — the pre-remediation 10 MVP drafts. They describe
  ambitious application shapes but used syntax/runtime features that current
  `main` did not parse, check, or run reliably.
